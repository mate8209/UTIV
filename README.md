# UT IV · Gobierno y Fallas de Mercado

Presentación web interactiva para GitHub Pages.

## Cómo publicar

1. Subir todos los archivos a un repositorio.
2. Verificar que `index.html` quede en la raíz.
3. Ir a **Settings → Pages**.
4. Elegir **Deploy from a branch**, rama `main`, carpeta `/root`.
5. Abrir la URL que entrega GitHub Pages.

## Uso

- Flecha derecha / izquierda: avanzar o retroceder.
- Botón **Notas** o tecla `N`: ver notas docentes.
- Botón **Índice** o tecla `T`: navegar por la estructura.

## Base

Material de cátedra: `UT 4 Gob y Fallas de Mercado 26.pdf`.
